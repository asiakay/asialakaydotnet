Project Samples

Make Music:
is a Single Page Application that enables the user to make live music. On the front-end, it uses ReactJS. And on the back-end uses Node.js, and Tone.js, a web audio framework for creating interactive music in the browser.
Watch Cryptos:
is a Single Page Application that allows the user to 1) select a cryptocurrency pair and 2) view a chart that displays the historical price up to the current real-time price. The back-end uses WebSockets to maintain the connection to the API that provides the data.
Nighttime Todos:
is a to-do list application made for viewing at night. The front-end is built with React and implements the ChakraUI component library. On the back-end, it manages the state and status of each to-do item using React Hooks.
AsiaLAKAYdotnet Blog:
is a customized WordPress blog that provides users with access to an updated event calendar, developer resources, and curated content centered around current events.
